    Changelog

    Welcome to the OnAire changelog. Here you can find out more about recent OnAire releases and what the changes mean for you as a OnAire user.

    March 12, 2021

    Version 0.1.2 OnAire goes multi tenant

    • Stopped local video from freezing when a remote participant video track changes
    • Host will no longer see ended message for guest when app is slow
    • Given side panels consistent styling
    • Added gallery view for host
    Feb 26, 2021

    Version 0.1.1 OnAire improvements

    • Uploaded files now display the name of the participant that uploaded them
    • File size is displayed with files that have been sent in chat
    • Guests view the host as main speaker and have a scrollable sideview of other participants
    • Host can admit all waiting participants
    • The host can remove guests from the meeting
    • Fixed an issue with end meeting
    • Guest is shown a message when the host has ended the meeting
    • Guest can access chat after the meeting has ended
    Feb 01, 2021

    Version 0.1.0 OnAire goes multi-participant

    • Multi-participant video calls
    • Multi-participant video chat
    • Waiting rooms
    • Preview a file that has been sent
